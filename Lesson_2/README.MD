# 基本 html 架構 _ 常用標籤(一)

-------------------------

## 段落、換行、標題：

* 段落標籤以及換行標籤



段落標簽 : `<p></p>`
換行標籤 : `<br>`

​	

什麼是段落標籤呢？就跟他的名稱一樣其實就是用來分段的
那換行標籤呢？就是換行

​	

以往我們寫文章在換行的時候都是按Enter就好，但是自己實際去寫會發現這在網頁裡根本沒用！

​	

在程式裡面的確是將這兩行文字換行了，但是在網頁裡他根本沒用，所以需要使用` <p></p>` 以及 `<br>`

-------------------------------

## 標題標籤

### h1 h2 h3 h3 h5 h6

那何謂標題標籤呢？
其實他們最大的差別就在於字體大小。

​	

h1最大，h6最小，但其實字體大小都是可是可以利用`CSS`去做修改的。

​	

那通常會怎麼使用呢？我們在網頁裡一定會有最重要和次要的標題對吧？
所以會利用重要性來選用，這裡就實際去示範。

​	

一樣是寫在`<body> </body> `裡，很簡單，結合目前的東西來做一個簡單的例子：

```html
<body>

  <h1>聲聲慢</h1>
  <h3>李清照</h3>
  <p>尋尋覓覓，冷冷清清，淒淒慘慘慼慼。
    <br> 乍暖還寒時候，最難將息。
    <br> 三杯兩盞淡酒，怎敵他、晚來風急？
    <br> 雁過也，正傷心，卻是舊時相識。
  </p>
  <p>滿地黃花堆積，憔悴損，如今有誰堪摘？
    <br>守著窗兒，獨自怎生得黑？
    <br>梧桐更兼細雨，到黃昏、點點滴滴。
    <br>這次第，怎一個、愁字了得！
  </p>

</body>
```
